# homebrew-talos

Homebrew tap for [Talos Works](https://talos.works) desktop tools.

## Setup

```sh
brew tap kivanccakmak/talos
```

## Apps

| App | macOS | Linux |
|-----|-------|-------|
| Talos Label Printer | `brew install --cask talos-label-printer` | `brew install talos-label-printer` |
| Talos Agent | `brew install --cask talos-agent` | — |
